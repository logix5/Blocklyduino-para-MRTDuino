BlocklyDuino Enhanced

Welcome to BlocklyDuino Enhanced

What is this tool?

BlocklyDuino Enhanced is a tool that generate an Arduino code from Blocks. No need to know about how to write arduino code and remember Arduino instruction

Who can use this tool?

  • Kids from the age of 7
  • Beginner with Arduino programming
  • Teacher who want to teach arduino in the class
  • Expert with other Microcontroller who have not the time to remember the syntax of Arduino

Who made this tool?

The tool was improved by Adel Kassah, a computer science teacher in Tunisia

This work is based on the work of Fred Lin (@gasolin) who created BlocklyDuino and Neil Fraser who created Blockly. I also took and corrected some blocks from www.technologiescollege.fr and used some feature of Blockly Lua.
